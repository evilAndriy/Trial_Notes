# Проект Trial_Notes


Проект Trial_Notes - Тестові замітки з додаванням, редагуванням,та вдаленням заміток реалізовані через ListAdapter


Для встановлення програми Trial_Notes, виконайте наступні кроки:

1.Відкрийте Android Studio. Виберіть опцію Get from VCS.
 Увійдіть в репозиторій на GitHub: https://github.com/evilAndriy/Trial_Notes

2.  У полі "URL" введіть URL-адресу вашого репозиторію GitHub : https://github.com/evilAndriy/Trial_Notes.git

3.  Продовжуйте створення проекту за потреби.

  Це дозволить вам вказати URL-адресу GitHub пі
